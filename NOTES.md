# Заметки по разработке

## Архитектура проекта

### Модули
- `config.py` - Конфигурация игры (константы, цвета, настройки)
- `entities.py` - Игровые объекты (Paddle, Ball, PowerUp)
- `game_state.py` - Управление состояниями игры (меню, игра, пауза)
- `audio.py` - Менеджер аудио (музыка, звуки)
- `effects.py` - Визуальные эффекты (частицы, trail, screen shake)
- `stats.py` - Статистика игр (сохранение в JSON)
- `settings.py` - Настройки игры (сохранение в JSON)
- `ui.py` - UI элементы (индикаторы, меню настроек)
- `pong_v4.py` - Главный файл игры

### Версии
- `pong.py` (v1.0) - Базовая версия для двух игроков
- `pong_v2.py` - Добавлен AI, power-ups, меню
- `pong_v3.py` - Уровни сложности, цветные ракетки
- `pong_v4.py` - Модульная архитектура, эффекты, статистика, настройки

## Управление

### Игрок 1
- `A` - Вверх
- `Z` - Вниз

### Общее
- `Enter` - Старт/Продолжить
- `Escape` - Пауза/Выход
- `S` - Статистика (в меню)
- `O` - Настройки (в меню)
- `1/2/3` - Выбор сложности (в меню)

## Технические детали

### Физика
- Векторная система движения мяча
- Угол отражения зависит от точки удара на ракетке
- Постепенное увеличение скорости мяча

### Эффекты
- Частицы создаются при ударе (10 частиц)
- Trail обновляется каждые 2 кадра
- Screen shake с затуханием интенсивности
- Анимация гола с fade out

### Сохранение данных
- `stats.json` - Статистика игр
- `settings.json` - Настройки пользователя
